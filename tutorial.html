Introdução

Bem-vindo ao painel de administração! Este guia irá orientá-lo através de todas as funcionalidades disponíveis para criar e gerenciar seus concursos de fotografia.

Passo 1: Gerenciamento de Usuários

A primeira etapa é garantir que os usuários corretos (avaliadores e outros administradores) estejam cadastrados no sistema.

  • Acesse a aba "Gerenciamento de Usuários" no painel principal.
  • Para registrar um novo usuário: Preencha o nome de usuário (sem espaços ou caracteres especiais), uma senha com no mínimo 6 caracteres e selecione a função:
    • Avaliador: Pode apenas visualizar e votar nas fotos dos concursos aos quais for atribuído.
    • Administrador: Tem acesso total ao sistema, podendo criar concursos e gerenciar usuários.
  • Para excluir um usuário: Na tabela "Usuários Cadastrados", clique no botão "Excluir" ao lado do usuário desejado. Você não pode excluir a si mesmo ou o administrador principal.

Passo 2: Gerenciamento de Concursos

Com os usuários cadastrados, você pode começar a criar os concursos.

  • Acesse a aba "Gerenciamento de Concursos".
  • Para criar um novo concurso: Preencha o nome e uma descrição opcional e clique em "Criar Concurso".
  • Gerenciar um concurso: Na lista de "Concursos Criados", você encontrará as seguintes opções para cada concurso:
    • Status: Altere o status do concurso para controlar seu ciclo de vida.
      • Rascunho: O concurso está sendo preparado. Avaliadores não podem vê-lo.
      • Aberto: O concurso está ativo e visível para os avaliadores designados votarem.
      • Fechado: A votação está encerrada. Os resultados ficam disponíveis na página de Ranking.
    • Editar: Permite alterar o nome e a descrição do concurso.
    • Gerenciar: Leva você para a página de detalhes do concurso, onde você adiciona categorias, fotos e avaliadores.
    • Excluir: Apaga permanentemente o concurso e todos os seus dados (fotos, categorias, avaliações). Esta ação não pode ser desfeita.

Passo 3: Detalhes do Concurso (Gerenciar)

Esta é a área onde você popula o concurso com conteúdo e define quem pode votar.

  • Aba "Categorias": Crie as categorias que organizarão as fotos (ex: "Paisagens", "Retratos"). Você pode editar o nome ou remover categorias a qualquer momento.
  • Aba "Fotos": Faça o upload das imagens para o concurso. Para cada foto, você deve selecionar um arquivo, associá-la a uma categoria e, opcionalmente, adicionar o nome do autor.
  • Aba "Avaliadores": Adicione os usuários (com função de "Avaliador") que terão permissão para votar nas fotos deste concurso específico. Um avaliador só verá os concursos aos quais for adicionado.

Passo 4: O Processo de Avaliação

Uma vez que um concurso está com o status "Aberto" e tem avaliadores designados, eles poderão fazer o login e avaliar as fotos. Eles verão uma interface para selecionar o concurso, a categoria e navegar entre as fotos, atribuindo notas para cada critério.

Passo 5: Visualizando os Resultados (Ranking)

Após o término da votação, altere o status do concurso para "Fechado".

  • Acesse a página de "Ranking" através do link no cabeçalho ou na página de login.
  • Selecione o concurso finalizado que deseja visualizar.
  • Você pode filtrar os resultados por categoria ou ver o ranking geral.
  • Use o botão "Exportar para CSV" para baixar uma planilha com os resultados da seleção atual.
